# Передумови (Вимоги до ПЗ)
Перед початком роботи переконайтеся, що на вашому комп'ютері встановлено:
- Python: версія 3.10 або вище.
- Git: для завантаження вихідного коду.
- pip: менеджер пакетів Python.

# Покрокова інструкція з розгортання (Роад-мапа)
## Крок 1. Клонування репозиторію
Отримайте локальну копію проєкту та перейдіть до робочої директорії :
```bash
git clone https://github.com/liaamirt/the-mix-bot.git
cd the-mix-bot
```

## Крок 2. Налаштування віртуального середовища
Створіть та активуйте ізольоване віртуальне середовище, щоб уникнути конфліктів глобальних залежностей :
```bash
# Створення віртуального середовища
python -m venv.venv

# Активація (для систем Windows):
.venv\Scripts\activate

# Активація (для систем Linux/macOS):
source.venv/bin/activate
Крок 3. Встановлення залежностей
Після активації середовища встановіть усі необхідні бібліотеки з файлу залежностей :

pip install -r requirements.txt
```

## Крок 4. Налаштування змінних оточення
Створіть файл .env у корені проєкту (цей файл автоматично ігнорується Git завдяки .gitignore для безпеки токенів). Налаштуйте змінні за зразком :

```
BOT_TOKEN=ваш_токен_від_BotFather
ADMIN_CHAT_ID=ідентифікатор_адміністратора
DB_PATH=themix.db
```
## Крок 5. Створення та ініціалізація бази даних
Структура бази даних (таблиці feedbacks та bookings) створюється автоматично під час першого запуску бота завдяки вбудованим SQL-скриптам. Якщо вам необхідно заповнити базу тестовими даними (бронюваннями та відгуками), виконайте допоміжний скрипт :

```bash
python seed_db.py
```
## Крок 6. Запуск проєкту
Для запуску Telegram-бота в режимі розробки виконайте головний файл :

```bash
python main.py
```

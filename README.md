The Mix Bot 🤖
Опис проєкту
Цей проєкт створено в рамках виконання лабораторної роботи 1 з дисципліни "Технічна підтримка програмного забезпечення". the-mix-bot — це Telegram-бот, розроблений мовою програмування Python. Особливістю архітектури є відмова від використання високорівневих фреймворків на користь "чистого" коду (Vanilla Python) та прямої інтеграції з Telegram Bot API через HTTP-запити.

Технологічний стек
Мова програмування: Python 3.9+

Взаємодія з API: використання бібліотеки requests (або вбудованих модулів urllib/http.client) для відправки мережевих запитів.

Управління залежностями: стандартний пакетний менеджер pip.

Встановлення та локальний запуск
**Клонування репозиторію:**bash
`git clone https://github.com/liaamirt/the-mix-bot.git`
`cd the-mix-bot`


Створення та активація віртуального середовища:

Bash
`python -m venv.venv`

# Для Windows:
.venv\Scripts\activate

Для macOS/Linux:
source.venv/bin/activate


3. **Встановлення залежностей:**
Якщо у проєкті використовується файл requirements.txt, виконайте:

`pip install -r requirements.txt`

Безпечне налаштування змінних оточення:
Створіть файл .env у кореневій директорії проєкту (він автоматично ігнорується системою Git для вашої безпеки). Додайте до нього ваш токен доступу до Telegram API:

Фрагмент кода
`TELEGRAM_TOKEN=ваш_секретний_токен_тут`

Запуск бота:

Bash
python main.py

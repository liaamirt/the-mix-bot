Структура цього файлу повинна містити такі розділи:
Пререквізити: Вимоги до наявності встановленого Python, активованого віртуального середовища та встановлених пакетів із requirements.txt , а також пакетів sphinx і sphinx-rtd-theme.
Процес оновлення джерел: Інструкція з використання команди sphinx-apidoc -o source/../, яка сканує вихідний код і автоматично створює або оновлює .rst файли для нових Python-модулів.
Процес збірки: Детальний опис виконання команди make html (або make.bat html для середовищ Windows) із вказівкою робочої директорії docs/.
Верифікація: Вказівка на місцезнаходження згенерованих HTML-артефактів (зазвичай це папка docs/_build/html) та інструкція з відкриття файлу index.html у локальному веб-браузері.

Архітектура проєкту
===================

Цей документ описує високорівневі архітектурні рішення Telegram-бота "The Mix".

Децентралізована маршрутизація
------------------------------
Проєкт відмовляється від монолітного Dispatcher на користь використання кількох незалежних об'єктів Router (`basic.py`, `booking.py`, `feedback.py`). Це патерн проектування гарантує ізоляцію контекстів та полегшує модульне тестування.

Асинхронна робота з БД
----------------------
Використання SQLite у асинхронному середовищі вимагає особливого підходу. Архітектура передбачає створення ізольованих транзакцій через асинхронні контекстні менеджери ``async with aiosqlite.connect(DB_PATH)``.
